# PureFlow Pressure Washing

Eco-friendly, water-only pressure washing website for driveways in Mount Juliet, TN and surrounding neighborhoods.

## Features

- **Compact Landing Page** — single-viewport, no-scroll layout with lead capture form
- **Booking Page** — calendar-based appointment scheduling
- **Gallery** — before/after photo showcase
- **Social Hub** — social media links and engagement
- **Quote Form** — instant free quote request
- **Service Areas** — weekly schedule across 6 neighborhoods
- **$99 Driveway Wash** — flat-rate pricing

## Tech Stack

- **React 19** with Vite
- **React Router** for navigation
- **Plain CSS** design system with CSS custom properties
- **Lucide React** for icons

## Getting Started

```bash
npm install
npm run dev
```

## Pages

| Route | Description |
|-------|-------------|
| `/` | Compact landing page with quote form |
| `/book` | Book a service appointment |
| `/social` | Social media hub |
| `/gallery` | Before/after photo gallery |

## Design

- Water-only, electric-powered, eco-friendly branding
- Blue/green color palette reflecting cleanliness and nature
- Mobile-first responsive layout
- Minimal, conversion-focused UI
